## Notes

### Content Strategy
- Front-load value (first 10 seconds critical)
- Always include clear call-to-action
- Reference community resources (Inner Circle, Plaza Group)
- Cross-promote between channels where relevant

### SEO Strategy
- Research keywords before creating content
- Optimize titles for search and click-through
- Use timestamps for longer videos
- Leverage playlists for series content

### Community Engagement
- Pin top comment with key resources
- Respond to comments within 24 hours
- Feature community questions in Q&A videos
- Acknowledge Inner Circle/Plaza Group members

## Calendar Management

### Adding New Videos
1. Choose target publish date
2. Add row to appropriate month
3. Set status: `idea -> planned -> research -> scripting -> draft -> edited -> final -> published`
4. Link to project folder once created

### Moving Dates
- Update this calendar first
- Update PROJECT.md `planned_publish_date`
- Adjust task deadlines in `03-Tasks/Tasks.md`

### Tracking Progress
- Update status column as videos move through production
- Update pipeline status counts monthly
- Review content balance quarterly
