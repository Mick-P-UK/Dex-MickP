## Key Concepts (Running Reference)

- **AX Tree**: The accessibility tree of a browser page - a structured
  representation of page elements (roles, names, states) used by screen
  readers and automation tools.
- **Playwright**: Browser automation library with MCP integration in Cowork.
- **Claude in Chrome**: Direct browser control via Claude's Chrome extension.
- **Accessibility Snapshot**: Playwright's `browser_snapshot` tool returns
  the AX tree of the current page.
