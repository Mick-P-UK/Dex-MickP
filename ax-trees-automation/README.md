# ax-trees-automation

## Overview

This folder contains research, scripts, and automation work related to
Accessibility Trees (AX Trees) - a browser automation and web-scraping
technique used within the PAIDA / Cedric ecosystem.

AX Trees allow Claude and other AI agents to interact with web pages
via the browser's accessibility layer rather than raw HTML, making
scraping and automation more robust and structured.

---

## Purpose

- Explore and document AX Tree-based scraping approaches
- Build reusable automation scripts and agents
- Store reference material, examples, and findings
- Track progress and decisions in memory.md
