## Decisions made

- **The Ron pattern is now the mandatory architecture** for the ShareScope + NBLM pipeline. Cedric orchestrates. Nina (NotebookLM) fetches and organises sources. Ron (a spawned sub-agent) reads the sources AND the chart PNG (using vision) and writes the analysis. Cedric receives Ron's report and saves it to the vault. This restores what the May 2026 webinar SOP described but the codified skill had shortcut by having `notebooklm ask` do the analysis via Nina.
- **Chart capture is now integrated into the pipeline as a mandatory step** (not just a nice-to-have). `sharescope_chart_orchestrator.py` produces the PNG, Ron reads it via vision. The chart is NOT uploaded to NBLM as a source (proven pointless - Nina cannot read it).
- **Chart period selector uses the "Other" dropdown + "1 year" text path**, not the bottom-panel preset buttons. Preset buttons named "12mth - Daily" are misleading - they are template configurations whose period is overridden by whichever top-panel tab (All / 2 years / Other) is currently active.
- **Tags for AI reports follow the format `[TICKER, Financial-Analysis, ShareScope-Research]`** - no spaces (Obsidian renders space-containing tags as invalid strikethrough). Ron-authored reports additionally carry a `Ron` tag.
- **Obsidian deep-links use the `vault=<name>&file=<relative-path>` format** on Windows, not `path=<absolute-path>`. The pipeline's current link generator uses the broken `path=` format - to be fixed in `sharescope_nlm_researcher.py` `_make_obsidian_uri()`. For manual links, `vault=Dex-MickP&file=06-Resources/Research-Log/Research/[TICKER]/[filename]` works reliably.
- **Sub-agent spawning is Cedric's tool for Ron.** For today's live run Cedric used `general-purpose` as the subagent type via the Agent tool. Whether Ron should live as a permanent named agent (own definition file) or continue to be spawned per-invocation from an inline prompt is deferred to tomorrow's SOP update discussion.
- **JSE report v2 (Ron-authored) supersedes v1 (Nina-authored)** in the JSE Research folder. Both retained; v2 is the current definitive report. ENQ still only has a v1 - regeneration deferred.
- **Level 1 autorun (Cedric runs the two Python scripts directly via PowerShell tool) is the new default** for the pipeline. Mick no longer needs to run terminal commands. Only the NBLM login refresh (`notebooklm login`) requires Mick's browser interaction, and that is only needed when auth expires (currently 316 days out).
