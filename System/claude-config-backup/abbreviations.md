# Personal Abbreviations

This file contains abbreviations and shorthand used across all Claude projects.

## Format
- **Abbreviation**: Full meaning (context/usage notes)

---

## Abbreviations

- **NBLM**: NotebookLM (Google's AI-powered note-taking and research tool)

---

## Usage

When you use an abbreviation Claude doesn't recognize, it will:
1. Check this file first
2. If not found, ask you for the meaning
3. Add it to this file for future reference

Use `/abbreviations` skill to:
- Add new abbreviations
- List all abbreviations
- Search for specific abbreviations
- Update existing entries
