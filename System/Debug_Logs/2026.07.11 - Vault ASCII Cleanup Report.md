# Vault ASCII Cleanup Report - 2026.07.11

Full non-ASCII sweep of the Dex-MickP vault, run in Cowork at Mick's request.
Root cause of the corruption: the original DEX templates were seeded on a Mac and
carried smart punctuation (curly quotes, em/en dashes, ellipses, non-breaking
spaces) that propagated into every note built from them.

## What was done

Stage 1 - Typography corruption (safe, automatic). Converted the Mac-origin set to
ASCII across every decodable text file in the vault:
- curly quotes -> straight quotes
- em dash / en dash -> hyphen
- ellipsis -> ...
- non-breaking / thin / zero-width spaces -> normal space or removed
- bullets, arrows, box-drawing (single and double) -> ASCII equivalents

Stage 2 - Meaningful characters (by Mick's decision, whole-vault scope):
- Currency: GBP for the pound sign, EUR for euro, c for cent, JPY, INR, KRW.
- Status glyphs to ASCII tags: tick -> [x], cross -> [ ], warning -> [!],
  up-triangle -> ^, down-triangle -> v, info -> [i], traffic lights -> [red]/
  [green]/[amber], etc.
- Decorative emoji: dropped.
- ShareScope private-use icon glyphs (captured in the AX-tree JSON scrapes): dropped.
- Broken U+FFFD replacement marks (bytes already lost): dropped.
- Accented letters: transliterated to their ASCII base (e.g. e-acute -> e).
- Nine wrong-encoding text files re-decoded from Windows-1252 and re-saved as clean
  UTF-8/ASCII, including four finished Writing System pieces (letters/newsletters).

## Result

- 60 files cleaned in the typography pass; 413 files cleaned in the full pass.
- Final verification: ZERO decodable text files contain any non-ASCII character.

## Files deliberately left untouched

- System/.credentials/google_calendar_token.pickle - binary credential (never touch).
- The .obsidian config folder - app-managed, not content.
- Six undecodable junk/artifact files (not live notes): two corrupted
  Week_Priorities .md.v3-backup backups, one test-webinar .log, two ShareScope
  report .tmp files, one Word ~WRL0005.tmp autosave. Safe to delete by hand if wanted.

## Prevention

The DEX templates (_templates, Micks-View/_templates, YouTube_System/_templates)
were cleaned in this pass, so new notes built from them start ASCII-clean. The
existing pre-commit ASCII hook continues to guard work-mcp inputs on commit.

## Git

Vault edits from this session ride the scheduled 9pm git sweep (single private repo,
SSH auto-push). All changes are therefore version-controlled and reversible.
