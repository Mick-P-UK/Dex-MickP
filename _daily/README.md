# _daily

Dated daily notes. Two kinds live here side by side, sharing the YYYY.MM.DD date
prefix so they sort together, distinguished by suffix so they never collide:

- YYYY.MM.DD - Journal.md   Mick's own daily journal (author: Mick).
- YYYY.MM.DD - Sundown.md   Cedric's end-of-day rollup (author: Cedric, AI-generated).

The Sundown note is written by the sundown-wrap skill and links to that day's Journal
note if one exists. Journal notes are never overwritten.

ASCII only for all files in this folder (vault pre-commit hook).
