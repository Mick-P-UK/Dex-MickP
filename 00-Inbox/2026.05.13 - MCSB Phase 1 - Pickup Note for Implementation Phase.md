## Architecture decisions to remember (top 10)

1. Source of truth: Dex vault (C:\Vaults\Mick's-Dex-2nd-Brain\Dex-MickP) - restructure in place, nothing destroyed.
2. Universal API: Cedric Server (FastAPI/FastMCP Python service) - NOT n8n.
3. Auth: two-tier bearer token in C:\Users\pavey\.env - PC token (full) / Mobile token (restricted).
4. Privacy: 99-Private/ (gitignored) + per-entry private: true frontmatter - defence in depth.
5. Ingestion: immutable raw->processed pattern - originals in 00-Inbox/raw/processed/ NEVER deleted.
6. agents.md: vault-root declarative rule file - loaded hourly by server; versioned in agents.md-history/.
7. Notion stays: Research Database, Companies Covered, Memory Vault all stay in Notion (bridge not migrate, D24).
8. Mobile sync: Working Copy (iOS) + Obsidian Git (Android) - free tier, no Obsidian Sync subscription.
9. CRM: Phase 5 - 02-People/ with strict naming, private: true mandatory, auto-index.
10. Journaling: Phase 5 - journal/ folder, trigger words journal/brainstorm/idea; distinct from brain dump captures.
