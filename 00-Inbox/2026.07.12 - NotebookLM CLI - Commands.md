# Title

## Key Points

## Listed NotebookLM CLI top-level commands

Here's what's actually available on your machine (fresh from `notebooklm --help`):

## Session

- `login` - authenticate via browser
- `use` - set the current notebook context
- `status` - show current context
- `clear` - clear notebook context

## Notebooks

- `list` - list all notebooks
- `create` - create a new notebook
- `delete` - delete a notebook
- `rename` - rename a notebook
- `summary` - AI-generated summary of a notebook

## Chat

- `ask` - ask the current notebook a question
- `configure` - configure chat persona / response settings
- `history` - get conversation history (or save it as a note)

## Source (`notebooklm source <cmd>`)

`add`, `add-drive`, `add-research`, `delete`, `delete-by-title`, `fulltext`, `get`, `guide`, `list`, `refresh`, `rename`, `stale`, `wait`

## Artifact (`notebooklm artifact <cmd>`)

`delete`, `export`, `get`, `list`, `poll`, `rename`, `suggestions`, `wait`

## Note (`notebooklm note <cmd>`)

`create`, `delete`, `get`, `list`, `rename`, `save`

## Share (`notebooklm share <cmd>`)

`add`, `public`, `remove`, `status`, `update`, `view-level`

## Research (`notebooklm research <cmd>`)

`status`, `wait`

## Generate (`notebooklm generate <type>`)

`audio` (podcasts), `cinematic-video`, `data-table`, `flashcards`, `infographic`, `mind-map`, `quiz`, `report`, `revise-slide`, `slide-deck`, `video`

## Download (`notebooklm download <type>`)

`audio`, `cinematic-video`, `data-table`, `flashcards`, `infographic`, `mind-map`, `quiz`, `report`, `slide-deck`, `video`

## Other

- `auth` - authentication management
- `language` - output language for generated artifacts
- `metadata` - export notebook metadata with sources list
- `skill` - manage Claude Code skill integration

---

Two things worth noting:

1. **`cinematic-video`** is on this install but wasn't in the skill's quick-reference table - a newer option worth trying if you want the fancier video style.
2. **`summary`, `configure`, and `metadata`** are also present that weren't in the skill's cheat-sheet - useful if you want an AI overview of a notebook, tweak the chat persona, or export a full sources list for a notebook.
