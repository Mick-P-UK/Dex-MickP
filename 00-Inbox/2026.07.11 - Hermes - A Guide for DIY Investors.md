# Hermes: The Open-Source AI Assistant You Own

DIY INVESTORS - AI-4-Investing

A plain-English guide for private investors. Prepared for the AI-4-Investing Webinar, 29 July 2026. diy-investors.ai

## 1. What is Hermes?

Hermes is an artificial intelligence assistant that lives on your own computer. It is free, open-source, and made by a company called Nous Research. The name covers two related things, and it helps to keep them apart.

The two Hermes:

- The Hermes models. These are the "brains" - AI language models in the same family as ChatGPT or Claude, but open-weight, meaning anyone can download them and run them on their own hardware. They are known for doing what you actually ask, with far less hedging, lecturing, or refusing.
- Hermes Agent. This is the part that has caused all the excitement. It is a personal AI assistant framework, released on 25 February 2026, that you install on your own machine. It has a memory that survives between sessions, it teaches itself new skills, it can run scheduled jobs while you sleep, and you can talk to it through Telegram, WhatsApp, Slack, email, or a plain terminal window.

The simplest way to think about it: imagine hiring a research assistant. Most AI chatbots are like hiring a very clever temp from an agency: brilliant for an afternoon, but every morning they arrive with total amnesia. Hermes Agent is like hiring a permanent member of staff. They remember your portfolios, your house style, your preferences. And when they work out a fiddly process, they write it down in a notebook (a "skill") so they never have to work it out again.

The kit-car analogy: if ChatGPT or Claude is a chauffeur-driven car, Hermes is a kit car. You own it, keep it in your own garage, look under the bonnet, tune it, and it goes exactly where you point it. That freedom is the appeal. It is also the risk.

Why everyone is suddenly talking about it: Hermes Agent went from nothing to over 140,000 GitHub stars in under three months, the fastest-growing open-source AI agent of the year. NVIDIA are promoting it heavily as the flagship reason to buy a powerful home PC, and it was demonstrated in Jensen Huang's GTC keynote.

Who is behind it? Nous Research started as a loose band of volunteers who met on Discord, GitHub and Twitter around 2022. Today it is a team of roughly twenty people. In April 2025 the crypto venture firm Paradigm led a 50 million US dollar funding round that valued the project at 1 billion US dollars.

Investor warning: there is no Hermes share to buy. Nous Research is a private company. Its billion-dollar valuation is a TOKEN valuation, not an equity valuation, and no official token had launched at the time of that raise. Any "NOUS" coin advertised on a crypto exchange should be treated as unofficial and almost certainly a scam.

## 2. Why Should You Consider Using It?

The case for:

1. It remembers, so it compounds. Hermes keeps a persistent memory of your work and, after it completes a fiddly task, writes itself a reusable skill file. It gets more useful the longer you use it - the shape of a compounding asset.
2. It works while you are not there. Hermes has a built-in scheduler. Tell it, in plain English, to check the LSE regulatory news for your holdings every weekday at 07:00 and message you on Telegram if anything material lands.
3. It comes to you, wherever you are. It runs across Telegram, WhatsApp, Slack, Signal, email and the terminal from a single gateway.
4. It is yours, and it is free. Open-source under the MIT licence. Your data, memory, skills and files stay on your machine. You can also run it against a completely local model, so nothing leaves the building.

The case against:

1. Security is a genuine, current issue. The biggest concern is "skill poisoning": because Hermes writes its own skills to disk and loads them later as trusted instructions, a single malicious instruction hidden in a web page it reads could get saved permanently and acted on days later ("prompt injection with a save button"). One disclosed vulnerability so far (CVE-2026-7396). The Cloud Security Alliance's advice: treat these agents as infrastructure, not as apps. Hermes ships with serious defences (blocklist, container sandboxing, credential stripping, website blocklist) but several of the strongest protections are opt-in.
2. It is not a beginner's tool - yet. You are still installing software that can read your files and run commands on your computer.
3. You still pay for the brain. The software is free; the AI model behind it usually is not. Either pay a provider or run a model locally (needs a decent graphics card).

## 3. How to Install It

Before you start: a computer (Windows, macOS or Linux - one-click installers on Windows and Mac); an AI provider (Nous Portal subscription, an API key, or a local model); model context of at least 64,000 tokens; about 20 minutes.

Route A: The Desktop installer (recommended). Download from hermes-agent.nousresearch.com, run the installer, open Hermes, choose Quick Setup with Nous Portal, and have a first conversation to verify.

Route B: Command line only.

- macOS/Linux/WSL2: curl -fsSL https://hermes-agent.nousresearch.com/install.sh | bash
- Windows PowerShell: iex (irm https://hermes-agent.nousresearch.com/install.ps1)
- Then: hermes setup --portal

Commands worth knowing: hermes (start chatting, add --tui); hermes setup; hermes model; hermes doctor; hermes skills browse; hermes gateway setup; hermes update; hermes --continue.

Safety switches to set before anything serious:

- Run it in a sandbox: hermes config set terminal.backend docker
- Never turn on YOLO mode (--yolo / /yolo switch off approval prompts). If you have, rotate your keys.
- Leave approvals on manual.
- Lock down who can message it (explicit allowlist of user IDs; never allow-all).
- Protect your keys file (~/.hermes/.env; chmod 600 on Mac/Linux; never sync to a public repo).
- Be careful what you feed it - skill poisoning arrives through content the agent reads.

A sensible first setup: install on a machine that does not hold your broker logins, run it in a Docker sandbox, point it at read-only data and your own files, get comfortable before giving it anything sensitive.

## 4. Ideas for Using It: The DIY Investor's Playbook

None of these are investment advice - they are ways to automate the donkey work.

Morning routine: the pre-market news sweep (regulatory news/announcements on each holding, summarised, sent to Telegram or email); the end-of-day movers report (rank today's moves, flag threshold breaches, add 5-day and YTD context).

Research and analysis: the company one-pager (financials, news, results, analyst consensus in a consistent house format); the results-season triage (compare numbers to prior year and guidance, highlight only what changed); your own research library (point Hermes at your notes, past reports, webinar transcripts).

Portfolio housekeeping: the weekly review pack (performance, week's news, stop-loss breaches, weighting drift); watchlist monitoring (target price, results, warnings); the year-end paperwork (broker CSVs, dividend schedules, transaction summaries).

For the ambitious: build a personal skills library (encode your investment process as reusable skills); test your own thinking (ask it to argue the bear case).

Where to be careful: never let it trade (information in, decisions and orders by you); verify before you act; it is not advice; watch what it ingests (keep allowlists tight).
