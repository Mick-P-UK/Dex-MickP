## Adding New Templates

When a new report type is needed (e.g. Newsletter, Portfolio Review), create a subfolder here with:
- The `.docx` template file
- A `README.md` describing placeholder fields and usage
- Reference logos from this shared `logos/` folder using `../logos/filename`
