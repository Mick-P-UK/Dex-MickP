# Week Plan Print - Colour Palette

## Primary Palette

| Token | Hex | Usage |
|-------|-----|-------|
| DARK_BLUE | #1F3864 | Day header backgrounds, title text |
| MID_BLUE | #2E5FA3 | Current day header, time column text |
| LIGHT_BLUE | #D6E4F0 | Accent highlights (optional) |
| ACCENT_ORANGE | #E67E22 | Reserved for urgent flags (future use) |
| LIGHT_GREY | #F2F2F2 | Alternating row shading |
| WHITE | #FFFFFF | Default row background |
| TEXT_DARK | #1A1A1A | Primary event text |
| TEXT_MED | #444444 | Secondary text |
| NOTE_GREY | #888888 | Inline notes / italics |
| FOOTER_GREY | #AAAAAA | Footer text |
| HEADER_DIM | #CCDDEE | Date text within day header |
| BORDER | #CCCCCC | Cell borders |

## Typography

- Font: Arial throughout
- Title: 36pt bold dark blue
- Date range in title: 28pt mid blue
- Day headers: 22pt bold white + 20pt dimmed date
- Time column: 18pt bold mid blue
- Event text: 18pt regular dark
- Notes (inline): 16pt italic note grey
- Footer: 14pt italic footer grey

## Design Principles

- Print-friendly: high contrast, no gradients
- Clean and readable at A4 pinned on a wall
- Today's day always highlighted in mid-blue
- Major event days (webinar, boot camp) noted in header
- Consistent with DIY Investors brand (blues + clean sans-serif)
