# _deprecated Skills

This folder holds retired PAIDA skills retained for reference and rollback purposes.

## Policy

- Skills are moved here (not deleted) when superseded by a newer version
- Each deprecated skill is renamed to include the version suffix (e.g. yt-weekly-stats-v1)
- A deprecation header is added to each SKILL.md recording: date, reason, and what superseded it
- This folder is NOT mirrored to /mnt/skills/user/ -- deprecated skills are invisible to Claude at runtime
- Review annually and archive to vault backup if no longer needed
